Loading a different map or set of cards

Loading a different map

Loading a different map only be done from the New Game screen, click on the Choose Map button and a smaller window should open, navigate to the map file and open it to load it. If the map file is valid, a mini picture should be displayed in the Map: window showing the map that you have selected, and the map file name will be displayed.

Valid maps supplied are:

luca.map (default map, map made by Luca)
ameroki.map (crazy map)
eurasien.map (Map of Europe and parts of Africa)
geoscape.map (geographical map of the world)
lotr.map (map from the game lotr risk)
risk.map (map from a old risk game)
RiskEurope.map (Map of Europe)
roman_empire.map (roman empire map)
sersom.map (supplied map)
teg.map (map from the game TEG)
tube.map (map of London Zone 1 Tube)
uk.map (alternative map requirement)
world.map (another geographical map of the world)

If the map is not valid, then the mini map will remain as it was, i.e. no change.

Loading a new set of cards

Valid cards supplied are:

ameroki.cards (cards that are used with ameroki.map)
eurasien.cards (cards that are used with eurasien.map)
lotr.cards (cards that are used with lotr.map)
nomission.cards (cards for uk.map, tube.map and others)
risk.cards (these are the standard cards)
RiskEurope.cards (cards that are used with RiskEurope.map)
roman_empire.cards (cards that are used with roman_empire.map)

Loading a set of cards can only be done form the New Game screen

Loading a different map only be done from the New Game screen, click on the Choose Cards button and a smaller window should open, navigate to the cards file and open it to load it, it will be displayed in the cards field.
