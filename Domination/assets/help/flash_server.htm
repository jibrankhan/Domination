Playing a game on the Internet or network

To start an Internet or network screen you need at least 2 computers, one computer will be the host i.e. the game server and the other computer(s) will join the server game.

To join an Internet or network game, the player needs only to click on the Join Game and enter either the computer name or Internet Protocol Number (IP) of the host computer and click Join Game. To find out your computer's IP or Name click the About button on the main screen

To start an Internet or network game follow the screen help.


Once in the game, the server game configuration screen is very much similar to a local game, there are some subtle differences compared to a local game however.

Warnings : Ensure ALL human players have joined the game BEFORE any players (either human or computer) are added, otherwise there will be problems with syncronisation and may lead to the game crashing.

In an Internet or network game, the Undo feature is disabled.

This is to prevent users from using the Undo function during network/internet games and gaining an unfair advantage.

Generally it is a good idea to let the host or a single person to adjust the game options and start the game.

It is also a good idea to use a communications program like GAIM to communicate between players.

The port used for network play is 4444.
